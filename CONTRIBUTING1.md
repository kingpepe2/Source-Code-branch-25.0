# Contributing to KingPepe

We welcome developers to contribute to **KingPepe**, a SHA-256 based meme coin originally based on **Bitcoin Core 25.0**.

Our current goal is to upgrade the full codebase to **Bitcoin Core 29.0**, while keeping all current parameters exactly the same.

## 💡 Coin Parameters to Preserve:
- Coin name: `KingPepe`
- Ticker: `KPEPE`
- Block reward: 50
- Halving interval: every 210,000 blocks
- Total supply: 22,000,000
- Network ports, address prefixes, and consensus logic

---

## 🛠️ What We Need:
- A clean port of the source to Bitcoin Core v29.0
- Full support for CLI/daemon and Qt GUI
- Build compatibility with Linux (Ubuntu 22.04) and optionally Windows

---

## 🤝 How to Contribute:
1. Fork the repository
2. Create a feature or upgrade branch
3. Submit a pull request with a clear description
4. Or open an issue if you'd like to discuss implementation
